# My Blog



## 简介
My Blog是由Docker+SpringBoot2.0+Mybatis+thymeleaf等技术实现的个人网站，是一个docker和springboot的实战练习项目.

- **你可以把它作为个人网站，集成了个人首页，个人摄影作品，个人博客等功能；**
- **你也可以把它作为springboot技术的学习项目，My Blog也足够符合要求，且代码和功能完备；**
